In Web-CAT's data model, your institution, "", can have one or more departments, and each department can have its own set of courses. In a given academic semester (or quarter), one or more sections of a course may be offered. In this installation step, you will identify the first department that will be created for your institution, the first semester in which courses will be offered, and an initial list of courses you want Web-CAT to know about.

If you have already installed Web-CAT and are working with a database that already contains this information, then leave the fields below blank. Only complete these fields if you want to create new database entries for the items listed below.

Create a semester/quarter
Leave this box checked and select dates below when you are setting up a brand new database. Uncheck this box to skip creation of a new semester object if you are using an existing database that already has a current semester created within it.
Period: Pick the label that most closely corresponds to the term you use for the semester/quarter you want to create.
Start date:   Select the starting date of the semester/quarter you wish to create. Dates are used for informational purposes only.
End date:   Select the ending date of the semester/quarter you wish to create. Dates are used for informational purposes only.
Create sample data
Check this box to automatically create a sample demo department called "Sandbox Department", which contains a demo course called "Sandbox Course". This may be useful if you are new to Web-CAT and are not yet ready to create your own department. You can change the name of this department and course any time you wish (as well as create more) using the administrator account. Soon we will include the creation of a couple of demo assignments for you to explore, but this part of the sample data is not yet ready.
Create your own department
Dept. name:
This is the name of the department in which you offer courses. For example, "Computer Science" or "Computer and Software Engineering". Leave this field blank if you do not wish to create a department data entry at this time. An administrator can create new department(s) later as needed, and you can have as many as you want.
Dept. abbrev.: This is the official abbreviation used by your university for identifying courses in the department named above. For example, "CS", or "COMPSCI".
Create a list of courses
If you would like to create your own list of courses within the department you have created above, list them here. All of these courses will be created within your new department (or within the Sandbox department if you have chosen to create a sample department rather than a new one of your own). List the course number without including your department's abbreviated name together with the course title. You may list up to five courses to create.
